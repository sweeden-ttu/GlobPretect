# Blooming-directed-graph agents

**Filter agents** (collections) and **trigger process agent** for the blooming-directed-graph. Run across all languages, projects, repositories, clusters, and models. See **docs/BLOOMING_DIRECTED_GRAPH_AGENTS.md** for the contract.

- `filter_agents/<platform>/` – filter nodes/edges by receiver, action_where, action_client, repo name, context key.
- `trigger_process_agent/<platform>/` – trigger workflow_dispatch, job_submit, git, run_script, notify.

Platforms: swift, objective-c, ruby, typescript, python, csharp, java, bash, zsh, git, github, hpcc.
